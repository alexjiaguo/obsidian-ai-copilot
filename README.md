# AI Copilot

**An AI assistant seamlessly integrated into your vault.**

This plugin brings Notion AI-like capabilities to your knowledge base, allowing you to summarize notes, brainstorm ideas, and chat with your files using your preferred LLM (OpenAI, Anthropic, Gemini, Groq, or Local).

## 🚀 Features

- **Context-Aware Chat**: Chat with your current note or your entire knowledge base.
- **Clean UI**: native-feeling interface with model and persona selectors.
- **Multiple Providers**: Support for OpenAI, Anthropic, Gemini, Groq, and Ollama.
- **Smart Mentions**: Use `@` to reference specific files in your chat.
- **Privacy-First**: Your data stays local (or is sent only to the API you choose).

## 🛠️ Installation

### Manual Installation
1.  Download the latest release (or build from source).
2.  Copy `main.js`, `manifest.json` to your plugin folder:
    `.obsidian/plugins/ai-copilot/`
3.  Reload.

### Build from Source
```bash
npm install
npm run build
```

## ⚙️ Configuration

1.  Open Settings > AI Copilot.
2.  Enter your API Key(s) for your preferred provider(s).
3.  (Optional) Configure custom Personas in the settings.
