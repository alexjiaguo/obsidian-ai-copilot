# Obsidian AI Copilot

**An AI assistant seamless integrated into your Obsidian vault.**

This plugin brings Notion AI-like capabilities to Obsidian, allowing you to summarize notes, brainstorm ideas, and chat with your vault using your preferred LLM (OpenAI, Anthropic, Gemini, Groq, or Local).

## 🚀 Features

- **Context-Aware Chat**: Chat with your current note or your entire vault.
- **"Notion-like" UI**: Clean, native-feeling interface with model and persona selectors.
- **Multiple Providers**: Support for OpenAI, Anthropic, Gemini, Groq, and Ollama.
- **Smart Mentions**: Use `@` to reference specific files in your chat.
- **Privacy-First**: Your data stays local (or is sent only to the API you choose).

## 🛠️ Installation

### Manual Installation
1.  Download the latest release (or build from source).
2.  Copy `main.js`, `manifest.json`, and `styles.css` to your vault's plugin folder:
    `.obsidian/plugins/obsidian_ai_copilot/`
3.  Reload Obsidian.

### Build from Source
```bash
npm install
npm run build
```

## ⚙️ Configuration

1.  Open Obsidian Settings > AI Copilot.
2.  Enter your API Key(s) for your preferred provider(s).
3.  (Optional) Configure custom Personas in the settings.
